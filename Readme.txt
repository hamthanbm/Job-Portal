To run this project:
	
	- Make sure Python is installed on your computer.

	- Open PowerShell or Command Prompt and navigate to the project location using the cd command.

	- Type pip install -r req.txt and press enter. Wait for all packages to be installed.

	- Type python manage.py runserver and press enter.
